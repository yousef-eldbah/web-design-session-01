# 🌐 NTI Web Design Track — Session 01

## Introduction to the Web & HTML Basics

Welcome to **Session 01** of the **NTI Web Design Track**.

This repository contains all the materials, examples, exercises, and project files covered in the first session of the Web Design Bootcamp.

---

## 📚 Session Overview

Before writing any code, every web developer should understand how the web works.

In this session, we explore the fundamentals of the Internet, how browsers communicate with servers, and how to build the structure of a web page using HTML5.

By the end of this session, students will be able to create their first complete semantic HTML page.

---

# 🎯 Learning Objectives

After completing this session, you should be able to:

* Understand how the Internet works
* Explain the Client–Server Architecture
* Understand DNS and HTTP/HTTPS
* Describe the Browser Rendering Process
* Understand the evolution of Web Design
* Apply modern design principles
* Install and configure Visual Studio Code
* Use Chrome DevTools for debugging
* Write valid HTML5 documents
* Create semantic HTML structures
* Work with headings, paragraphs, images, links, and lists
* Speed up development using Emmet

# 📌 Best Practices

* Use semantic HTML whenever possible.
* Keep your code clean and properly indented.
* Use only one `<h1>` per page.
* Always include the `alt` attribute for images.
* Use meaningful filenames.
* Validate your HTML code regularly.
